MarkovTextGenerator.py -help

Usage: MarkovTextGenerator.py [options]

Options:
  -h, --help            show this help message and exit
  -m MODE, --mode=MODE  mode: train or generate
  -o ORDER, --order=ORDER
                        order of Markov chain (only in train mode)
  -f FILE, --file=FILE  file name to store trained chain in train mode or
                        restore in generate mode
  -u URLS, --url=URLS   url which contains text, can be set multiple times
                        (only in train mode)
  -w WORDS, --word=WORDS
                        beginning of sentence (only in generate mode)
  -s SIZE, --size=SIZE
                        size of generated sequence (only in generate mode)

Examples:
MarkovTextGenerator.py -m train -o 3 -f dump.cPickle -u https://raw.githubusercontent.com/mephistopheies/ml-r/master/Python/MarkovChainTextGenerator/big.txt
MarkovTextGenerator.py -m generate -f dump.cPickle -s 100 -w когда -w нравственно
